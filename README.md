# 现代任务清单网站

一个使用Tailwind CSS构建的现代化、响应式任务清单网站。

## 功能特性

- ✅ 添加新任务并设置优先级（高/中/低）
- ✅ 标记任务完成状态
- ✅ 编辑和删除任务
- ✅ 筛选任务（全部/待完成/已完成）
- ✅ 统计数据展示
- ✅ 任务数据本地存储
- ✅ 一键导出任务数据为JSON
- ✅ 响应式设计，支持移动端
- ✅ 美观的毛玻璃效果界面

## 技术栈

- **HTML5** - 语义化标记
- **Tailwind CSS** - 现代化CSS框架
- **JavaScript (ES6)** - 交互逻辑
- **LocalStorage** - 数据持久化
- **Font Awesome** - 图标库

## 使用方法

1. 直接打开 `index.html` 文件即可运行
2. 输入任务内容并选择优先级
3. 点击"添加任务"或按回车键
4. 使用筛选按钮查看不同状态的任务
5. 点击任务前的复选框标记完成
6. 使用编辑/删除按钮管理任务

2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为源

## 自定义

### 修改主题颜色
编辑 `index.html` 中的渐变颜色：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
